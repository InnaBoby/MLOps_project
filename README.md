# MLOps_project

Data source: https://www.kaggle.com/datasets/ferraz/cotatenis-sneakers?resource=download-directory

##Setup

Склонируйте локально текущий репозиторий

```
git clone https://github.com/InnaBoby/MLOps_project.git
```

Для создания виртуального окружения проекта используется poetry. Активируйте виртуальное окружение командой

```
eval $(poetry env activate)
```
